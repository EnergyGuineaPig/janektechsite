jeasydata:

how to import:

download the file using
pip install Jeasydata==0.1.3
After that just use
import jeasydata

This addon will be updated someday(if i want to). for now you need use this. You can modify the addon and if you want to publish it just please add that the base was from janektech.pl

Functions:

encode. Encode function adds a bite to the file so other reading functions won't read it 
file = "file.jbk"
jeasydata.encode(file)
decode. Decode function deletes the first bite from the file 
file = "file.jbk"
jeasydata.decode(file)
Read. Read function reads the file provided and returns dataout with is a list 
file = "file.jbk"
out = jeasydata.read(file)
print(out)
Result:  ['sigma', '@tuff boy', 'sigma2', '@tuff boy2']
target. Target function is used when you now the first value.[apple]: like; it will return like 
file = "file.jbk"
out = jeasydata.target("sigma",file)
print(out)
Result:  ['tuff boy']
readAll. readAll function is used when you don't know the target. it will return all of the targets and also data 
file = "file.jbk"
out = jeasydata.readAll(file)
print(out)
Result:  ['sigma', 'sigma2'] ['tuff boy', 'tuff boy2']
readAllTargets. readAllTargets function is used when you don't know the target and you want to read and save data one by one from them. it will return all of the targets and also data 
file = "file.jbk"
out = jeasydata.readAllTargets(file)
print(out)
Result:  ['sigma', 'sigma2'] ['tuff boy', 'tuff boy2']
{'sigma': ['tuff boy'], 'sigma2': ['tuff boy2']}
Save. saves files :). Val1 is the target and Val2 is the data 
file = "file.jbk"
val1 = "apple"
val2 = "mango"
jeasydata.save(val1, val2, file)
Result:  saved file
Delete. deletes the targeted line in the file :). Val1 is the target 
file = "file.jbk"
val1 = "apple"
jeasydata.delete(val1, file)
Result:  deleted apple and its value in the file

Functions:

init. init is used to create the tempfile. use it at the start of your program *importing*
import jprinter as j
j.init()
Result:  temprint.html created
set. set is used to change the font, font color(for title or for normal text). use it at the start of your program ex. 1 
j.set("font", "Tahoma")
Result:  font for the normal text has been set to Tahoma
ex. 2 
j.set("Titlefont", "Tahoma")
Result:  font for the title has been set to Tahoma
ex. 3 
j.set("Titlecolor", "red")
Result:  Title color has been changed to red
ex. 4 
j.set("Fontcolor", "blue")
Result:  Font color has been changed to blue
addtext. addtext is used to add a line of text 
j.addtext("i visited janektech.pl !")
Result:  Created a new line of text in the file
addtitle. addtitle is used to add title 
j.addtitle("My document")
Result:  Created a new Title in the file
addphoto. addphoto is used to add a photo to the file. do NOT forget about r before your path. 
j.addphoto(r"c:/photo.jpg")
Result:  Added a photo to the file
print. it prints the file 
j.print()
Result:  prints the file using the default printer
end. it deletes the temp file 
j.end()
Result:  file deleted
